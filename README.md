# Página Web con Secciones de Artículos

Este proyecto consiste en una **página web básica** desarrollada con **HTML y CSS**, cuyo objetivo es mostrar distintos artículos organizados en secciones.

La estructura de la página utiliza etiquetas semánticas como `section` y `article`, permitiendo una correcta organización del contenido. Cada artículo contiene un título y un párrafo de texto.

El diseño visual se realiza mediante CSS, donde a cada sección se le asigna un color diferente con el fin de identificar y diferenciar visualmente cada artículo.

El proyecto tiene como propósito reforzar el uso de la estructura HTML y la aplicación de estilos básicos con CSS en una página web sencilla.

Proyecto realizado con fines académicos.
